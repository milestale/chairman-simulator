# Описание
Это симулятор создания стеклянных стульев сделанный для показа работы GameMaker с базами данных.

Сама игра была полностью написана на GameMaker Language, являющийся C подобным языком. Для связи с базой данных использовались PHP скриптыЮ так как движдок не имеет нативного способа управлять базами данных, но нетворкинг составляюющая достаточно развита.

База данных SQLite была развёрнута на MAMP сервере, так как он имеет user friendly интерфейс, что упростило разработку.

Данный проект демонстрирует работу онлайн сохранений, управлением аккаунтом и вход в систему с использованием базы данных.

## Предварительная настройка
Для запуска проекта вам потребуется какой-нибудь SQL сервер, например MAMP (Именно его я и использовал).

Запускайте сервер и в Preferences  выставите порт 81.
![img](/screenshots/postselect.png)

После этого настройте свою папку с документами, после чего перенесите все файлы из db. Сревер можете выбрать какой хотите.
![docs](/screenshots/server.png)

Незабудьте выставить версию PHP 5.7.24
![php](/screenshots/phpversion.png)

Теперь, можете заходить в phpliteadmin и наблюдать за работой игры!

## Сама игра

После захода игры вас встретит титульный экран, на котором можно произвести вход или регистрацию (Кнопки не имеют анимаций, но всё равно работают).

![Title](/screenshots/titlescreen.png)

После входа вы можете приступать к производству стульев
![WENEEDTOCOOKJESSE](/screenshots/breakingbad.jpg).

Игра выполнена в изометрическом стиле, где вы можете ходить по комнате, чтобы собирать разные предметы требуемые для создания стеклянного стула. Процесс побора выполнен в виде трёх небольших мини игр, где мы кладём песок в котёл, нагреваем его, чтобы получилось жидкое стекло и заливаем его в формочку. По окончанию создания вашего первого стула, вам покажут его в анимации и поздравят. После этого игра продолжается бесконечно и вы можете продоавать свои стулья, зарабатывая на этом монеты.

![s1](/screenshots/game.png)
![s2](/screenshots/minigame1.png)
![s3](/screenshots/minigame2.png)
![s4](/screenshots/minigame3.png)
![s5](/screenshots/ending.png)

Во время игры вы можете нажать на свой юзернейм, после чего вылезет небольшое флай аут меню, где можно перейти в настройки аккаунта или выйти из него.

![flyout](/screenshots/accountflyout.png)

В настройках вы можете выбрать себе кастомный аватар (Скейлинг не был настроен, поэтому для корректного отображения используйте картинки разрешения 563x545 пикселей), сменить никнейм и пароль, а также удалить свой аккаунт (ВАЖНО: после удаления аккаунта освобождается ваш юзернейм, но не ID, поэтому если вы передумали и зарегистрируете свой аккаунт заново, то ваш ID будет изменён на новый).

![settings1](/screenshots/changedavatar.png)
