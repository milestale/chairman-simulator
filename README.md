# Описание
Это симулятор создания стеклянных стульев сделанный для показа работы GameMaker с базами данных.

Сама игра была полностью написана на GameMaker Language, являющийся C подобным языком. Для связи с базой данных использовались PHP скриптыЮ так как движдок не имеет нативного способа управлять базами данных, но нетворкинг составляюющая достаточно развита.

База данных SQLite была развёрнута на MAMP сервере, так как он имеет user friendly интерфейс, что упростило разработку.

Данный проект демонстрирует работу онлайн сохранений, управлением аккаунтом и вход в систему с использованием базы данных.

## Предварительная настройка
Для запуска проекта вам потребуется какой-нибудь SQL сервер, например MAMP (Именно его я и использовал).

Запускайте сервер и в Preferences  выставите порт 81.
![Порты].scr/screenshots/portselection.png

В папку sqlite перенесите все файлы из db. После чего смените адрес и порт сервера на http://localhost:81.
